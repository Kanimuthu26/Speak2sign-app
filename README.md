# Speak2Sign App

Speak2Sign is a web-based communication system designed to improve accessibility for hearing and speech-impaired individuals. The application converts spoken input into sign representations and enables more inclusive communication.

## Features

- Speech to sign language conversion
- Real-time interaction
- Accessible web interface
- Supports inclusive workplace communication

## Technologies Used

- HTML
- CSS
- JavaScript
- React (via CDN)
- MediaPipe

## How to Run the Project

1. Download or clone the repository
2. Open `index.html` in any web browser
3. Start interacting with the application
